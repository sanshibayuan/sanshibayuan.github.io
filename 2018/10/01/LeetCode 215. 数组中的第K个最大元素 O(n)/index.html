Leetcode 215 数组最大间隔（O(n)复杂度实现）Python

发表于 2018-10-01 |

LeetCode 地址：https://leetcode-cn.com/problems/kth-largest-element-in-an-array/description/

题目描述

在未排序的数组中找到第 k 个最大的元素。

如:
输入: [3,2,1,5,6,4] 和 k = 2
输出: 5

要求用线性时间复杂度实现。

题目分析

线性时间复杂度实现，就说明不能先对元素进行排序，因为采用此法复杂度为O(n⋅log(n))

但我们可以借鉴快排算法中的Partition操作来完整题目。

Partition操作

由于题目中要求找到的是第k大的数，因此此处的Partition操作应该实现从大到小的划分。

每执行一次Partition操作，都将数组划分成比基准大和比基准小的两部分。

def partition_reverse(self,seq,left,right):
    array = left-1
    pivot_index = right
    pivot = seq[right]
    for j in range(left,right):
        if seq[j]>=pivot:
            array += 1
            seq[j],seq[array] = seq[array],seq[j]
    seq[array+1],seq[pivot_index] = pivot,seq[array+1]
    return array+1

当我们执行Partition操作返回的数值正好等于k-1，即说明有k-1个元素比基准值大，这说明所选基准值是第k大的元素，即为所求。当返回值大于k-1，则说明我们选择的基准值过小，此时应在[left,pivot-1]范围内寻找新的基准值；反之，则在[pivot+1,right]这个区间内寻找。

完整代码

class Solution(object):
    def findKthLargest(self, nums, k):
        """
        :type nums: List[int]
        :type k: int
        :rtype: int
        """
        return self.search(nums,0,len(nums)-1,k)

    def search(self,nums,left,right,k):
        pivot = self.partition_reverse(nums,left,right)
        if pivot == k-1:
            return nums[pivot]
        elif pivot > k-1:
            return self.search(nums,left,pivot-1,k)
        elif pivot < k-1:
            return self.search(nums,pivot+1,right,k)

    def partition_reverse(self,seq,left,right):
        array = left-1
        pivot_index = right
        pivot = seq[right]
        for j in range(left,right):
            if seq[j]>=pivot:
                array += 1
                seq[j],seq[array] = seq[array],seq[j]
        seq[array+1],seq[pivot_index] = pivot,seq[array+1]
        return array+1
